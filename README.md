# SALOON

A luxury grooming, styling, and beauty parlour web application.
This project contains:
- Elegant, premium UI styling with matte charcoal, deep obsidian, and warm metallic gold color themes.
- Rejuvenating services: Signature Haircuts, Beard Designs, Facial Therapies, and Groom Makeovers.
- Client reviews and responsive contact/booking options.
